# GodEtter 
GodEtter is a "smart fridge" WoT project made for the IDG3006 course at NTNU Gjøvik. 

It is built using:
*  **React** (front-end)
* **Python** (hardware / Pi)
* **PHP + MySQL** (back-end).

## Getting Started
To set this project up locally, check the README files in all the individual folders.
